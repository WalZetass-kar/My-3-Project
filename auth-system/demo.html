Struktur Data (localStorage)

authData

Menyimpan credentials admin (username & password)

isAuthenticated

Status login: true atau false

currentUser

Info user yang sedang login (username, login time)

landingPageData

Semua konten landing page (hero, about, contact)

⚠️ Penting!

Data disimpan di browser localStorage. Jika clear browser data, semua akan hilang. Gunakan fitur Export untuk backup!

Cara Menggunakan

1. Login ke Dashboard

Klik tombol "Login Page" → Masukkan username & password → Klik "Masuk"

2. Edit Konten Landing Page

Di dashboard, isi form dengan konten yang diinginkan. Bisa edit text, upload gambar, dll.

3. Upload Gambar (Optional)

Klik area upload → Pilih gambar (max 5MB) → Preview akan muncul

4. Simpan Perubahan

Klik tombol "Simpan Perubahan" → Data tersimpan ke localStorage

5. Lihat Hasil

Buka "Landing Page" → Konten yang diedit akan muncul!
